# AutoStudy-Logger

This repository maintains a consistent learning streak by automatically logging daily study sessions. 

## How it works
- **Automation:** A GitHub Action runs daily at 08:00 UTC.
- **Log:** It appends a timestamped entry to `daily_notes.md`.
- **Goal:** To maintain professional consistency and track my journey as an aspiring AI Engineer.

## Customization
To manually trigger the update, go to the **Actions** tab, select **AutoStudy-Logger**, and click **Run workflow**.
